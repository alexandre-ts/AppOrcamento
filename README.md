# AppOrcamento

Este trabalho consiste no desenvolvimento de um aplicativo de orçamento com o objetivo de auxiliar no controle de receitas e despesas. O sistema permite o cadastro, a organização e a análise dos dados financeiros, proporcionando uma melhor visualização dos gastos. O projeto aplica conceitos estudados em sala de aula, contribuindo para o desenvolvimento de soluções práticas voltadas à organização financeira.
